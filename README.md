# middleware test

## 主要功能
藉由middleware讓開發者在畫面載入時可以查看後台server log
### 顯示數據
- 載入時間
- 載入方法
- 方法網址
- 載入所花費時間

## 編程環境
Node.js v10.24.1

## 使用框架及套件
- express 4.17.1
- moment.js 

## 安裝流程
1. git clone本專案下載至本地
```
https://github.com/Chao69/middleware.git
```
2. 進入專案資料夾
```
cd middleware
```
3. 安裝套件
```
npm install express
npm install --save moment
```
5. 啟動網頁伺服器
```
npm run dev
```
6. 啟動成功後，即可造訪網站測試
-  App running on port 3000 
http://localhost:3000
